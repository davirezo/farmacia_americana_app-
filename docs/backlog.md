# Backlog de Épicos Iniciais

## [Épico 01] Identidade e Segurança (Auth & LGPD)
* Cadastro de usuários com validação de CPF.
* Gestão de termos de consentimento (LGPD) para dados de saúde.
* Recuperação de senha e perfil do paciente.

## [Épico 02] Catálogo e Experiência de Venda (E-commerce)
* Vitrine inteligente com algoritmos de recomendação (Sazonalidade).
* Navegação por categorias e busca com autocompletar.
* Carrinho de compras e fluxo de checkout simplificado.

## [Épico 03] Inteligência de Atendimento (IA & Chat)
* Interface de chat em linguagem natural.
* Módulo de triagem para medicamentos isentos de prescrição (MIPs).
* Geração de protocolos de atendimento para o farmacêutico.

## [Épico 04] Visão Computacional e Receitas
* Módulo de captura de foto para receitas.
* Integração com IA para OCR e validação de validade/prescritor.
* Trava de segurança para medicamentos controlados.

## [Épico 05] Fidelização e Predição (Ciclo de Saúde)
* Dashboard de histórico de compras.
* Algoritmo de predição de término de estoque pessoal.
* Sistema de notificações push "Reminder de Recompra".
